# Projeto de Aprendizado ThreeJS e GSAP

Este é um repositório de aprendizado dedicado ao estudo de animações 3D no React com ThreeJS e GSAP. O conteúdo é baseado no tutorial em vídeo "React 3D Animation Website Tutorial with ThreeJS (WebGL) & GSAP".

## Objetivos

O objetivo principal deste projeto é aprender e entender os conceitos fundamentais de ThreeJS e GSAP e como eles podem ser usados para criar animações 3D impressionantes em um site React.

## Progresso

Este repositório será atualizado regularmente com base no progresso do tutorial em vídeo.

## Recursos

O principal recurso utilizado para este projeto de aprendizado é o tutorial em vídeo "React 3D Animation Website Tutorial with ThreeJS (WebGL) & GSAP".

## Execução do Projeto

Instruções sobre como baixar e executar este projeto localmente serão fornecidas em breve.
